Приложение по поиску автомобилей.
Тестирование стоит автоматизировать так как приложение рассчитано на длительную работу и основная функция приложения состоит в поиске новых и Б/У автомобилей с различными фильтрами в поиске.
Тест-кейсы которые стоит автоматизировать:
1.	Тесты для регистрации.
2.	Тесты для авторизации.
3.	Тесты для поиска объявлений.
4.	Тесты для добавления объявлений.
5.	Тесты для удаления объявлений.

Тест-кейсы которые не стоит автоматизировать:
1.	Тесты удобства пользования.
2.	Тесты пользовательского интерфейса.
3.	Тесты новой функциональности.
4.	Тесты сложных пользовательских путей с большим количеством шагов.
5.	Тесты с оплатой.
